# Design Spec: High-Integrity Markowitz Efficient Frontier

## Goal
Replace the current randomized 'cloud' with a mathematically proven Efficient Frontier Line and Opportunity Cloud generated by the industry-standard **PyPortfolioOpt** (Python) library.

## Architecture: The Polyglot Bridge
We will implement a **Python-to-TypeScript Bridge** to ensure zero math slop.
1. **Next.js (Server):** Gathers historical return data from the Simba dataset for your specific constituents.
2. **Python Script (`src/lib/logic/math/optimizer.py`):** 
   - Takes the return data via JSON.
   - Calculates the **Ledoit-Wolf Shrinkage Covariance Matrix**.
   - Solves the Quadratic Programming problem to find 20 optimal points on the Frontier.
   - Generates 500 valid randomized permutations for the 'Opportunity Cloud' context.
3. **TypeScript Service:** Executes the Python script using `child_process` and parses the resulting optimal coordinates.

## Mathematical Requirements
- **Risk Model:** Ledoit-Wolf Shrinkage (reduces noise in historical covariance).
- **Optimization:** Maximize Sharpe Ratio and Minimize Volatility for 20 target return increments.
- **Constraints:** 
  - Weight Sum = 1.0.
  - Long-Only ($w_i \ge 0$).
  - Asset Universe: Limited to the user's current holdings and target strategy.

## UI/UX Integration
- **The Line:** A solid, surgical line showing the **Optimal Frontier**.
- **The Cloud:** 500 stable, mathematically valid permutations (the 'Possible' universe).
- **The Points:** 
  - **Market (VTI)**: White dot.
  - **Strategy (Target)**: Indigo dot.
  - **Portfolio (Actual)**: Rose dot.
- **The Gap:** Visual distance between 'Actual' and the 'Line' is clearly explained in the 'Forensic Takeaway' sidebar.

## Delivery Phases
1. **Phase 1 (Python Core):** Setup minimal virtual environment and the solver script.
2. **Phase 2 (The Bridge):** Implement the Node.js execution layer.
3. **Phase 3 (Visualization):** Update the Efficiency Map to render the stable Line + Cloud.
